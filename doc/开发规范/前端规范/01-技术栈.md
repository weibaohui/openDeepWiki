# 技术栈

本文档定义 openDeepWiki 前端项目使用的技术栈和版本要求。

## 核心技术

| 技术         | 版本   | 用途       |
| ------------ | ------ | ---------- |
| React        | 18.x   | UI 框架    |
| TypeScript   | 5.x    | 类型安全   |
| Vite         | 5.x    | 构建工具   |
| Tailwind CSS | 4.x    | 原子化样式 |
| shadcn/ui    | latest | UI 组件库  |
| React Router | 6.x    | 路由管理   |
| Axios        | 1.x    | HTTP 请求  |
| Lucide React | latest | 图标库     |

## 国际化与主题

| 技术          | 版本  | 用途       |
| ------------- | ----- | ---------- |
| i18next       | 23.x  | 国际化框架 |
| react-i18next | 14.x  | React i18n |
| next-themes   | 0.3.x | 主题切换   |

## 版本选择原则

1. **稳定性优先**：使用 LTS 或稳定版本
2. **兼容性考虑**：确保依赖间版本兼容
3. **安全性更新**：及时更新安全补丁
