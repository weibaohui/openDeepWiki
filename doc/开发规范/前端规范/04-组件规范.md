# Ant Design 使用规范

本文档定义 Ant Design 组件库的使用方式和最佳实践。

本文示例不包含代码注释，原示例注释已移除，如需保留请自行补充。

## 组件引用

```tsx
import { Button, Card, Input, Table, Badge, Skeleton, Modal, Form, Space } from 'antd';
import { PlusOutlined } from '@ant-design/icons';
```

## 常用组件

| 组件      | 用途     | 示例                                       |
| --------- | -------- | ------------------------------------------ |
| Button    | 按钮     | `<Button type="primary">提交</Button>`     |
| Card      | 卡片容器 | `<Card title="标题">内容</Card>`           |
| Input     | 输入框   | `<Input placeholder="请输入" />`           |
| Form      | 表单     | `<Form layout="vertical">...</Form>`       |
| Table     | 表格     | `<Table columns={columns} dataSource={data} />` |
| Modal     | 对话框   | `<Modal open={open}>...</Modal>`           |
| Drawer    | 抽屉     | `<Drawer open={open}>...</Drawer>`         |
| Tabs      | 标签页   | `<Tabs items={items} />`                   |
| Skeleton  | 骨架屏   | `<Skeleton active />`                      |
| Badge     | 徽章标签 | `<Badge count={5} />`                      |

## Button 使用场景

```tsx
<Space>
  <Button type="primary">主要操作</Button>
  <Button>次要操作</Button>
  <Button type="dashed">补充操作</Button>
  <Button type="text">文本操作</Button>
  <Button type="link">链接操作</Button>
  <Button danger>危险操作</Button>
</Space>
```

## 表单使用示例

```tsx
import { Button, Form, Input } from 'antd';

function RepoForm() {
  const [form] = Form.useForm();

  return (
    <Form form={form} layout="vertical">
      <Form.Item name="name" label="仓库名称" rules={[{ required: true }]}>
        <Input placeholder="请输入仓库名称" />
      </Form.Item>
      <Form.Item name="url" label="仓库地址" rules={[{ required: true, type: 'url' }]}>
        <Input placeholder="https://github.com/..." />
      </Form.Item>
      <Form.Item>
        <Button type="primary">提交</Button>
      </Form.Item>
    </Form>
  );
}
```

## Modal 使用示例

```tsx
import { Button, Modal } from 'antd';
import { useState } from 'react';

function DeleteConfirm() {
  const [open, setOpen] = useState(false);

  return (
    <>
      <Button danger onClick={() => setOpen(true)}>删除</Button>
      <Modal
        title="确认删除"
        open={open}
        onOk={() => setOpen(false)}
        onCancel={() => setOpen(false)}
      >
        删除后无法恢复，请谨慎操作。
      </Modal>
    </>
  );
}
```

## Table 使用示例

```tsx
import { Table } from 'antd';

const columns = [
  { title: '名称', dataIndex: 'name' },
  { title: '状态', dataIndex: 'status' },
];

const data = [
  { key: '1', name: 'openDeepWiki', status: 'ready' },
];

function RepoTable() {
  return <Table columns={columns} dataSource={data} pagination={false} />;
}
```

## 全局配置

```tsx
import { ConfigProvider } from 'antd';

function AppConfig({ children }: { children: React.ReactNode }) {
  return <ConfigProvider>{children}</ConfigProvider>;
}
```
