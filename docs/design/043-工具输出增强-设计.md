## 一、需求概述

为 read_file 与 search_file 工具输出补充行号；升级 list_dir 工具，默认过滤常见目录并支持 .gitignore / .dockerignore 规则过滤，以提升工具输出质量与可用性。

## 二、技术设计

### 2.1 read_file 行号输出

- 在读取文件内容时，按照真实文件行号拼接输出
- offset/limit 仅控制读取范围，行号仍对应原文件

### 2.2 search_file 行号输出

- 搜索文件时记录匹配行号
- 每条结果包含路径、行号与匹配内容

### 2.3 list_dir 过滤规则

- 内置默认忽略目录列表（.git、node_modules、.idea、.vscode、dist、build、vendor）
- 启动时读取仓库根目录下 .gitignore / .dockerignore，并合并为忽略规则
- 过滤规则应用于目录遍历与输出阶段
