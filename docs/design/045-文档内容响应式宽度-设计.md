# 1. 总体方案
在 DocViewer 页面内容容器上引入响应式最大宽度策略，基于 Ant Design 的断点信息动态设置 maxWidth，同时保持 width: 100% 与居中布局。

# 2. 前端设计

## 2.1 容器策略
- 使用 useBreakpoint 获取断点
- 内容容器设置 width: 100%
- 根据断点设置 maxWidth：
  - xxl：1400
  - xl：1200
  - lg：1000
  - 其他：不设上限（100%）

## 2.2 影响范围
- DocViewer 内容区域（包含概览、编辑与阅读模式）
- 不调整 Sidebar 与 Header 布局

# 3. 兼容性与安全
- 不新增后端接口与数据结构
- 不引入新的第三方依赖
