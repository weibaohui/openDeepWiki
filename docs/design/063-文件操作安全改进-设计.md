## 1. 需求背景

当前系统为大模型提供了文件操作工具（read_file、list_dir、search_files、run_terminal_command 等），用于在代码仓库分析过程中读取文件内容、执行命令等操作。但是，目前的安全检查存在以下问题：

1. **路径安全检查不完善**：`isPathSafe` 函数使用字符串前缀匹配，存在被绕过的风险（如使用 `../` 或符号链接）
2. **终端命令工具未限制工作目录**：`run_terminal_command` 允许通过 `working_dir` 参数指定任意目录作为工作目录，可能访问系统敏感文件
3. **缺乏统一的安全验证入口**：各工具各自实现路径检查，容易遗漏

## 2. 设计目标

1. **强制限制文件操作范围**：所有文件操作（读、写、列目录、搜索、执行命令）只能发生在目标代码仓库的本地目录内
2. **防止路径遍历攻击**：阻止使用 `../`、符号链接等方式访问基础目录之外的文件
3. **统一安全验证机制**：建立统一的路径安全验证工具，所有文件操作必须通过该验证

## 3. 核心设计

### 3.1 路径安全验证增强

**当前实现问题分析**：

```go
// 当前的 isPathSafe 实现
func isPathSafe(basePath, targetPath string) bool {
    absBase, _ := filepath.Abs(basePath)
    absTarget, _ := filepath.Abs(targetPath)
    absBase = filepath.Clean(absBase)
    absTarget = filepath.Clean(absTarget)

    if absTarget == absBase {
        return true
    }
    // 问题：字符串前缀匹配可能被绕过
    return strings.HasPrefix(absTarget, absBase+string(filepath.Separator))
}
```

**问题场景**：
- `/safe/base` 和 `/safe/base_backup` 情况下，`/safe/base_backup/secret.txt` 会被错误地认为在基础目录内

**改进方案**：

```go
// 增强的 isPathSafe 实现
func isPathSafe(basePath, targetPath string) bool {
    // 1. 获取绝对路径
    absBase, err := filepath.Abs(basePath)
    if err != nil {
        return false
    }
    absTarget, err := filepath.Abs(targetPath)
    if err != nil {
        return false
    }

    // 2. 清理路径（移除 . 和 ..）
    absBase = filepath.Clean(absBase)
    absTarget = filepath.Clean(absTarget)

    // 3. 解析符号链接（关键改进）
    realBase, err := filepath.EvalSymlinks(absBase)
    if err != nil {
        realBase = absBase // 如果解析失败，使用原始路径
    }
    realTarget, err := filepath.EvalSymlinks(absTarget)
    if err != nil {
        realTarget = absTarget
    }

    // 4. 确保基础目录以分隔符结尾，避免前缀匹配问题
    if !strings.HasSuffix(realBase, string(filepath.Separator)) {
        realBase += string(filepath.Separator)
    }

    // 5. 检查目标路径是否等于或在基础目录内
    if realTarget == filepath.Clean(realBase) {
        return true
    }

    // 6. 使用分隔符确保精确匹配目录边界
    return strings.HasPrefix(realTarget+string(filepath.Separator), realBase)
}
```

### 3.2 终端命令工具安全加固

**当前问题**：`run_terminal_command` 允许任意指定 `working_dir`，可能访问系统任意目录。

**改进方案**：

1. **强制验证工作目录**：执行命令前验证 `working_dir` 必须在 `basePath` 内
2. **禁止危险命令**：限制或禁止可能访问外部路径的命令（如 `cd / && cat /etc/passwd`）
3. **命令白名单机制**（可选）：只允许执行安全的命令前缀

```go
func (t *RunTerminalCommandTool) InvokableRun(ctx context.Context, argumentsInJSON string, opts ...tool.Option) (string, error) {
    // ... 参数解析 ...

    // 确定工作目录
    workingDir := t.WorkingDir
    if args.WorkingDir != "" {
        workingDir = args.WorkingDir
    }

    // 新增：验证工作目录安全性
    if workingDir != "" && !isPathSafe(t.WorkingDir, workingDir) {
        return "", fmt.Errorf("working directory escapes base directory: %s", workingDir)
    }

    // 新增：验证命令不包含路径遍历
    if containsPathTraversal(args.Command) {
        return "", fmt.Errorf("command contains path traversal sequences")
    }

    // ... 执行命令 ...
}

// 检查命令是否包含路径遍历
func containsPathTraversal(command string) bool {
    dangerousPatterns := []string{
        "../", "..\\",
        "`", "$()",
    }
    cmd := strings.ToLower(command)
    for _, pattern := range dangerousPatterns {
        if strings.Contains(cmd, pattern) {
            return true
        }
    }
    return false
}
```

### 3.3 统一的安全验证函数

在 `path_utils.go` 中增加以下函数：

```go
// ValidateAndResolvePath 验证并解析目标路径
// 返回：是否安全、解析后的绝对路径、错误信息
func ValidateAndResolvePath(basePath, inputPath string) (bool, string, error) {
    // 清理输入路径
    fullPath := filepath.Join(basePath, inputPath)
    if filepath.IsAbs(inputPath) {
        fullPath = inputPath
    }

    // 验证路径安全
    if !isPathSafe(basePath, fullPath) {
        return false, "", fmt.Errorf("path escapes base directory: %s", inputPath)
    }

    // 解析最终路径
    absPath, err := filepath.Abs(fullPath)
    if err != nil {
        return false, "", fmt.Errorf("failed to resolve path: %w", err)
    }

    return true, absPath, nil
}

// ValidateWorkingDir 验证工作目录
func ValidateWorkingDir(basePath, workingDir string) error {
    if workingDir == "" {
        return nil
    }
    if !isPathSafe(basePath, workingDir) {
        return fmt.Errorf("working directory escapes base directory: %s", workingDir)
    }
    return nil
}
```
