# 文档展示显示仓库地址 - 实现总结

## 1. 功能概述
在文档展示页顶部区域增加仓库地址显示，帮助用户快速确认文档来源，保持现有编辑与导出功能不变。

## 2. 需求对应关系
| 需求点 | 实现情况 | 说明 |
| :--- | :--- | :--- |
| **显示仓库地址** | ✅ 已实现 | 文档标题下方新增仓库地址行 |
| **支持中英文** | ✅ 已实现 | 新增 `document.repository_url` 文案 |
| **移动端可展示** | ✅ 已实现 | 保持响应式布局并支持省略展示 |
| **失败不影响文档** | ✅ 已实现 | 仓库请求失败不阻断文档展示 |

## 3. 关键实现点
### 3.1 文档展示页数据获取
- 在 DocViewer 中新增仓库详情获取逻辑，独立于文档加载流程。

### 3.2 页面展示与国际化
- 在文档标题区域追加仓库地址展示行。
- 新增中英文文案键值以保持国际化一致性。
