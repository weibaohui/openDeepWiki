## 1. 功能概述
在文档展示页内容区域增加仓库地址显示，帮助用户快速确认文档来源；在文档概览页新增最近更新标题列表，展示最新 5 条文档标题并可快速跳转。

## 2. 需求对应关系
| 需求点 | 实现情况 | 说明 |
| :--- | :--- | :--- |
| **显示仓库地址** | ✅ 已实现 | 文档内容区域顶部新增仓库地址行 |
| **支持中英文** | ✅ 已实现 | 新增 `document.repository_url` 文案 |
| **移动端可展示** | ✅ 已实现 | 保持响应式布局并支持省略展示 |
| **失败不影响文档** | ✅ 已实现 | 仓库请求失败不阻断文档展示 |
| **最近更新标题展示** | ✅ 已实现 | 概览页展示最新 5 条标题与空状态 |

## 3. 关键实现点
### 3.1 文档展示页数据获取
- 在 DocViewer 中新增仓库详情获取逻辑，独立于文档加载流程。

### 3.2 页面展示与国际化
- 在文档内容区域顶部追加仓库地址展示行。
- 新增中英文文案键值以保持国际化一致性。

### 3.3 最近更新标题展示
- 使用文档列表按更新时间倒序排序并截取 5 条。
- 标题使用链接按钮展示并跳转至对应文档页。
- 无数据时显示空状态文案。

## 5. 已知限制与待改进点
