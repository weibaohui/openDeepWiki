## 1. 实现概述
新增文档概览页面，首页仓库卡片点击进入概览页，同时提供“任务管理”入口跳转至任务管理页。概览页左侧展示文档列表，右侧展示文档与任务统计信息。

## 3. 关键实现点

### 3.1 新增文档概览页面
- 新增页面 `RepoDocumentIndex`，复用 Drawer/Sider 结构展示文档列表。
- 聚合任务状态与文档版本统计并通过 Statistic 展示。

### 3.2 路由与首页入口调整
- 增加 `/repo/:id/index` 路由入口。
- 首页仓库卡片点击跳转文档概览，新增“任务管理”按钮直达任务页。

### 3.3 国际化补齐
- 新增文档概览相关文案与空态文案。

## 5. 安全自检

1. **权限风险**：未新增后端接口，仅复用已有读接口。
2. **数据泄露**：未输出敏感信息，接口返回数据不做额外日志打印。
3. **输入处理**：路由参数仅用于 ID 查询，未引入新的用户输入面。
