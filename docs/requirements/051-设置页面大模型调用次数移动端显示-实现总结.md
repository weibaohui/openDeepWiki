## 4. 关键实现点

1. **问题定位**：通过分析代码定位到 `responsive: ['lg']` 属性导致列在移动端不显示
2. **解决方案**：移除该属性，使列在所有屏幕尺寸下都显示
3. **兼容性保障**：表格已配置横向滚动（`scroll={{ x: 800 }}`），新增列不会破坏布局
