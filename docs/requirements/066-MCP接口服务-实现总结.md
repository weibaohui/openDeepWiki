## 1. 实现概述

成功实现了 MCP (Model Context Protocol) 接口服务，使 AI 编程工具（如 Cursor、Claude Code、Windsurf）能够通过 MCP 协议查询项目文档。

### 1.1 实现的功能

| 功能 | 状态 | 说明 |
|------|------|------|
| list_repositories | ✅ 已完成 | 列出所有可用仓库 |
| get_repository | ✅ 已完成 | 获取仓库详情和文档列表 |
| search_documents | ✅ 已完成 | 搜索文档内容 |
| read_document | ✅ 已完成 | 读取文档完整内容 |
| SSE 传输 | ✅ 已完成 | 支持互联网访问 |

## 4. MCP 工具说明

### 4.1 list_repositories

列出所有可用的代码仓库。

**参数**：无

**返回示例**：
```json
{
  "repositories": [
    {
      "id": 1,
      "name": "openDeepWiki",
      "url": "https://github.com/xxx/openDeepWiki",
      "status": "completed",
      "branch": "main",
      "document_count": 10,
      "updated_at": "2026-03-16T10:00:00Z"
    }
  ]
}
```

### 4.2 get_repository

获取仓库详情，包含该仓库下的所有文档列表。

**参数**：
- `repo_id` (可选): 仓库ID
- `repo_name` (可选): 仓库名称（与 repo_id 二选一）

**返回示例**：
```json
{
  "id": 1,
  "name": "openDeepWiki",
  "url": "https://github.com/xxx/openDeepWiki",
  "branch": "main",
  "commit": "abc123",
  "status": "completed",
  "documents": [
    {
      "id": 1,
      "title": "概览",
      "filename": "overview.md",
      "sort_order": 1
    }
  ]
}
```

### 4.3 search_documents

搜索文档内容。通过关键词在文档标题、文件名、内容中进行匹配搜索。

**参数**：
- `query` (必需): 搜索关键词
- `repo_id` (可选): 限定仓库ID，不传则搜索所有仓库

**返回示例**：
```json
{
  "results": [
    {
      "doc_id": 1,
      "repo_id": 1,
      "repo_name": "openDeepWiki",
      "title": "架构设计",
      "filename": "architecture.md",
      "snippet": "...这是关于架构设计的..."
    }
  ],
  "total": 1
}
```

### 4.4 read_document

读取文档的完整内容（Markdown 格式）。

**参数**：
- `doc_id` (必需): 文档ID

**返回示例**：
```json
{
  "id": 1,
  "repo_id": 1,
  "repo_name": "openDeepWiki",
  "title": "架构设计",
  "filename": "architecture.md",
  "content": "# 架构设计\n\n## 概述\n...",
  "branch": "main",
  "commit": "abc123",
  "version": 1
}
```

## 5. 使用方式

### 5.1 启动服务

MCP 端点随主服务启动，无需额外配置：

```bash
cd backend
make build
./server
```

### 5.2 连接端点

MCP SSE 端点：`http://localhost:8080/mcp/sse`

### 5.3 AI 工具配置示例

**Claude Code** 配置示例：
```json
{
  "mcpServers": {
    "openDeepWiki": {
      "url": "http://localhost:8080/mcp/sse"
    }
  }
}
```
