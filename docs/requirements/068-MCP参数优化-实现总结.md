## 4. 工具变更详情

### 4.1 list_repositories
```go
// 新增参数
mcp.WithNumber("limit", "默认 20，最大 100")
mcp.WithNumber("offset", "从 0 开始")
mcp.WithString("status", "ready/pending/failed/cloning")

// 返回新增字段
{
  "repositories": [...],
  "total": 150,
  "has_more": true,
  "limit": 20,
  "offset": 0
}
```

### 4.2 get_repository
```go
// 优化描述
"优先使用 repo_id 查询，如不知道 ID 可使用 repo_name"

// 新增参数
mcp.WithBoolean("include_content", "默认 false")

// 返回控制
// include_content=false: 仅返回文档列表（id, title, filename）
// include_content=true: 额外返回 content 字段
```

### 4.3 search_documents
```go
// 新增参数
mcp.WithNumber("limit", "默认 10，最大 50")
mcp.WithString("doc_type", "api/architecture/guide/readme")

// 返回新增字段
{
  "results": [...],
  "total": 45,
  "has_more": true,
  "limit": 10
}
```

### 4.4 get_document_summary (新增)
```go
// 参数
mcp.WithNumber("doc_id", "必需")

// 返回
{
  "id": 1,
  "title": "文档标题",
  "summary": "前 500 字内容...",
  "total_length": 5000,
  "has_full_content": true
}
```

## 5. AI 使用流程优化

### 优化前
```
1. list_repositories → 获取全部仓库（可能很大）
2. get_repository (id=1) → 获取所有文档详情
3. read_document (id=123) → 读取完整内容（可能 Token 超限）
```

### 优化后
```
1. list_repositories (limit=10) → 分页获取仓库
2. get_document_summary (id=123) → 快速判断相关性
3. read_document (id=123) → 确认相关后读取完整内容
```
