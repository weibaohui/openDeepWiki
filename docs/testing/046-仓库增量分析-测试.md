## 测试范围

- 仓库详情页菜单按钮展示

- 增量变更日志输出

## 测试用例

### 用例 1：按钮展示

- 前置条件：进入仓库详情页面
- 操作步骤：打开“更多操作”抽屉
- 预期结果：出现“仓库增量分析”按钮

### 用例 2：接口成功调用

- 前置条件：仓库记录存在，本地路径可用，base commit 存在
- 操作步骤：点击“仓库增量分析”按钮
- 预期结果：接口返回成功，后台日志打印增量变更内容

### 用例 3：仓库不存在

- 前置条件：repoID 不存在
- 操作步骤：调用接口
- 预期结果：接口返回错误，后台日志记录原因

### 用例 4：本地路径不存在

- 前置条件：仓库记录存在但本地路径失效
- 操作步骤：调用接口
- 预期结果：接口返回错误，后台日志记录原因

### 用例 5：git pull 失败

- 前置条件：本地仓库存在但拉取失败（模拟断网或远端异常）
- 操作步骤：调用接口
- 预期结果：接口返回错误，后台日志记录原因
