## 三、集成测试

### 3.1 端到端测试流程

#### 测试场景：文档保存后自动生成向量

1. 创建一个仓库
2. 创建一个文档
3. 验证 VectorTask 被创建（status: pending）
4. 等待任务处理完成
5. 验证 DocumentVector 被创建
6. 验证向量被添加到索引
7. 执行语义搜索，验证文档被找到

#### 测试代码示例

```go
func TestVectorGenerationE2E(t *testing.T) {
    if testing.Short() {
        t.Skip("skipping e2e test")
    }

    // Setup
    db := setupTestDB()
    router := setupRouter(db)
    defer cleanupTestDB(db)

    // Step 1: 创建仓库
    repoID := createTestRepository(t, router)

    // Step 2: 创建文档
    docID := createTestDocument(t, router, repoID, "Test Title", "Test Content")

    // Step 3: 验证任务创建
    time.Sleep(100 * time.Millisecond)
    tasks := getPendingTasks(t, db)
    assert.Greater(t, len(tasks), 0)

    // Step 4: 等待任务完成
    waitForTaskCompletion(t, db, tasks[0].ID)

    // Step 5: 验证向量创建
    vector, err := getDocumentVector(t, db, docID)
    assert.NoError(t, err)
    assert.NotNil(t, vector)

    // Step 6: 验证搜索结果
    results := searchVectors(t, router, "Test Title")
    assert.Greater(t, len(results), 0)
    assert.Equal(t, docID, results[0].DocumentID)
}
```
