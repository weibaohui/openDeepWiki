## 1. 测试概述

本文档定义 AI 助手移动端响应式适配功能的 E2E 测试用例，使用 `playwright-cli` 工具执行。

- 后端服务已启动（http://localhost:8080）

|----------|----------|------|

## 3. 测试用例

### 3.1 桌面端默认显示测试

**测试目的**：验证桌面端默认显示 AI 助手

**前置条件**：桌面端视口（1920x1080）

**执行命令**：

```bash
# 1. 打开页面
playwright-cli open http://localhost:3000/#/repo/1/index

# 2. 设置桌面端视口
playwright-cli resize 1920 1080

# 3. 等待页面加载
sleep 2

# 4. 获取快照验证 AI 助手存在
playwright-cli snapshot

# 5. 验证 AI 助手可见（查找包含"AI"或"助手"的元素）
playwright-cli eval "document.body.innerText.includes('AI') || document.body.innerText.includes('助手')"

# 6. 截图保存
playwright-cli screenshot --filename=/tmp/test_desktop_default.png

# 7. 关闭浏览器
playwright-cli close-all
```

**预期结果**：
- AI 助手面板默认显示
- 包含"AI 助手"或"AI 文档助手"文本

---

### 3.2 移动端默认隐藏测试

**测试目的**：验证移动端默认隐藏 AI 助手，只显示按钮

**前置条件**：移动端视口（375x667）

**执行命令**：

```bash
# 1. 打开页面
playwright-cli open http://localhost:3000/#/repo/1/index

# 2. 设置移动端视口
playwright-cli resize 375 667

# 3. 等待页面加载
sleep 2

# 4. 获取快照
playwright-cli snapshot

# 5. 验证 AI 按钮存在
playwright-cli eval "document.querySelector('button[title*=\"AI\"], button[class*=\"ant-btn\"]') !== null"

# 6. 验证 AI 助手面板不存在（不包含"AI 助手"标题）
playwright-cli eval "!document.body.innerText.includes('AI 助手')"

# 7. 截图保存
playwright-cli screenshot --filename=/tmp/test_mobile_default.png

# 8. 关闭浏览器
playwright-cli close-all
```

**预期结果**：
- 仅显示 AI 助手按钮（机器人图标）
- AI 助手面板默认不显示

---

### 3.3 移动端点击 AI 按钮全屏显示测试

**测试目的**：验证移动端点击 AI 按钮后 AI 助手全屏显示

**前置条件**：移动端视口（375x667），AI 助手默认关闭

**执行命令**：

```bash
# 1. 打开页面
playwright-cli open http://localhost:3000/#/repo/1/index

# 2. 设置移动端视口
playwright-cli resize 375 667

# 3. 等待页面加载
sleep 2

# 4. 获取快照并找到 AI 按钮
playwright-cli snapshot
grep "button.*robot" .playwright-cli/page-*.yml | tail -1

# 5. 点击 AI 按钮（假设按钮 ref 为 e15）
playwright-cli click e15

# 6. 等待动画完成
sleep 1

# 7. 验证 AI 助手全屏显示
playwright-cli eval "document.body.innerText.includes('AI 助手')"

# 8. 验证输入框存在
playwright-cli eval "document.querySelector('input, textarea') !== null"

# 9. 截图保存
playwright-cli screenshot --filename=/tmp/test_mobile_fullscreen.png

# 10. 关闭浏览器
playwright-cli close-all
```

**预期结果**：
- AI 助手占满整个屏幕
- 显示"AI 文档助手"标题
- 显示输入框
- 文档内容不可见

---

### 3.4 移动端关闭 AI 助手测试

**测试目的**：验证移动端可正常关闭 AI 助手

**前置条件**：移动端视口，AI 助手已打开

**执行命令**：

```bash
# 1. 打开页面
playwright-cli open http://localhost:3000/#/repo/1/index

# 2. 设置移动端视口
playwright-cli resize 375 667

# 3. 等待页面加载
sleep 2

# 4. 点击 AI 按钮打开
playwright-cli snapshot
grep "button.*robot" .playwright-cli/page-*.yml | tail -1
playwright-cli click e15
sleep 1

# 5. 获取快照找到关闭按钮（X 按钮）
playwright-cli snapshot

# 6. 点击关闭按钮（假设关闭按钮 ref 为 e195）
playwright-cli click e195

# 7. 等待动画完成
sleep 1

# 8. 验证 AI 助手已关闭
playwright-cli eval "!document.body.innerText.includes('AI 助手')"

# 9. 验证文档内容可见
playwright-cli eval "document.body.innerText.includes('文档概览')"

# 10. 截图保存
playwright-cli screenshot --filename=/tmp/test_mobile_close.png

# 11. 关闭浏览器
playwright-cli close-all
```

**预期结果**：
- AI 助手面板关闭
- 文档内容重新显示
- AI 按钮可见

### 3.5 响应式切换测试

**测试目的**：验证窗口大小变化时正确切换 AI 助手显示模式

**前置条件**：从移动端切换到桌面端

**执行命令**：

```bash

# 2. 设置移动端视口

sleep 2

# 3. 点击打开 AI 助手

# 4. 切换到桌面端视口
playwright-cli resize 1920 1080
sleep 2

# 5. 验证 AI 助手仍然可见

# 6. 截图保存
playwright-cli screenshot --filename=/tmp/test_responsive_switch.png

# 7. 关闭浏览器
playwright-cli close-all
```

**预期结果**：
- 视口切换后 AI 助手保持打开状态
- 布局正确适应新视口

## 4. 测试脚本整合

### 4.1 完整测试脚本

```bash
#!/bin/bash
# 065-ai-assistant-mobile-responsive-test.sh
# AI 助手移动端响应式适配 E2E 测试脚本

set -e

echo "=== AI 助手移动端响应式适配测试 ==="

# 测试 1: 桌面端默认显示
echo "[TEST 1] 桌面端默认显示 AI 助手"
playwright-cli open http://localhost:3000/#/repo/1/index
playwright-cli resize 1920 1080
sleep 2
playwright-cli screenshot --filename=/tmp/065_test1_desktop.png
result=$(playwright-cli eval "document.body.innerText.includes('AI') || document.body.innerText.includes('助手')" 2>/dev/null || echo "false")
if [ "$result" = "true" ]; then
    echo "✓ 测试通过：桌面端 AI 助手默认显示"
else
    echo "✗ 测试失败：桌面端 AI 助手未显示"
fi
playwright-cli close-all

# 测试 2: 移动端默认隐藏
echo "[TEST 2] 移动端默认隐藏 AI 助手"
playwright-cli open http://localhost:3000/#/repo/1/index
playwright-cli resize 375 667
sleep 2
playwright-cli screenshot --filename=/tmp/065_test2_mobile_default.png
result=$(playwright-cli eval "!document.body.innerText.includes('AI 助手')" 2>/dev/null || echo "false")
if [ "$result" = "true" ]; then
    echo "✓ 测试通过：移动端 AI 助手默认隐藏"
else
    echo "✗ 测试失败：移动端 AI 助手不应默认显示"
fi
playwright-cli close-all

# 测试 3: 移动端点击全屏
echo "[TEST 3] 移动端点击 AI 按钮全屏显示"
playwright-cli open http://localhost:3000/#/repo/1/index
playwright-cli resize 375 667
sleep 2
playwright-cli snapshot > /dev/null 2>&1
# 查找 AI 按钮并点击
btn_ref=$(grep -o "button.*robot.*ref=e[0-9]*" .playwright-cli/page-*.yml 2>/dev/null | grep -o "e[0-9]*" | head -1)
if [ -n "$btn_ref" ]; then
    playwright-cli click "$btn_ref"
    sleep 1
    playwright-cli screenshot --filename=/tmp/065_test3_mobile_fullscreen.png
    result=$(playwright-cli eval "document.body.innerText.includes('AI 助手')" 2>/dev/null || echo "false")
    if [ "$result" = "true" ]; then
        echo "✓ 测试通过：移动端 AI 助手全屏显示"
    else
        echo "✗ 测试失败：移动端 AI 助手未全屏显示"
    fi
else
    echo "✗ 测试失败：未找到 AI 按钮"
fi
playwright-cli close-all

echo "=== 测试完成 ==="
```
