# 技术栈

本文档定义 openDeepWiki 前端项目使用的技术栈和版本要求。

## npm包管理

本项目使用pnpm 作为npm包管理工具。
一切依赖都通过pnpm 安装和管理。
一切命令都通过pnpm 执行。

## 核心技术

| 技术         | 版本   | 用途       |
| ------------ | ------ | ---------- |
| React        | 18.x   | UI 框架    |
| TypeScript   | 5.x    | 类型安全   |
| Vite         | 5.x    | 构建工具   |
| Ant Design   | 5.x    | UI 组件库  |
| React Router | 6.x    | 路由管理   |
| Axios        | 1.x    | HTTP 请求  |
| Ant Design Icons | 5.x | 图标库     |

## 国际化与主题

| 技术          | 版本  | 用途       |
| ------------- | ----- | ---------- |
| Ant Design Locale | 5.x | 组件多语言 |
| Ant Design Theme  | 5.x | 主题定制   |

## 版本选择原则

1. **稳定性优先**：使用 LTS 或稳定版本
2. **兼容性考虑**：确保依赖间版本兼容
3. **安全性更新**：及时更新安全补丁
