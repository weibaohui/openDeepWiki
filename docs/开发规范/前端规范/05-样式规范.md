# Ant Design 样式规范

本文档定义 Ant Design 的样式与布局规范，强调使用内置布局系统与主题变量。

本文示例不包含代码注释，原示例注释已移除，如需保留请自行补充。

## 响应式要求

所有页面必须使用响应式布局，确保在不同屏幕尺寸下正常显示。

## 布局与间距

优先使用 Ant Design 的布局组件和间距体系。

```tsx
import { Col, Row, Space } from 'antd';

function LayoutSample() {
  return (
    <Space direction="vertical" size={16} style={{ width: '100%' }}>
      <Row gutter={[16, 16]}>
        <Col span={12}>左侧</Col>
        <Col span={12}>右侧</Col>
      </Row>
    </Space>
  );
}
```

## 排版与内容

使用 Typography 统一字体层级与颜色。

```tsx
import { Typography } from 'antd';

const { Title, Text } = Typography;

function TypoSample() {
  return (
    <>
      <Title level={3}>页面标题</Title>
      <Text type="secondary">辅助说明</Text>
    </>
  );
}
```

## 主题变量与 Token

样式定制优先使用 Token，避免硬编码颜色。

```tsx
import { theme } from 'antd';

function TokenSample() {
  const { token } = theme.useToken();

  return (
    <div style={{ background: token.colorBgContainer, padding: token.padding }}>
      <span style={{ color: token.colorText }}>内容区域</span>
    </div>
  );
}
```

## 表单与按钮样式

按钮类型与表单布局优先使用组件能力，不新增外部样式库。

```tsx
import { Button, Form, Input } from 'antd';

function FormSample() {
  return (
    <Form layout="vertical">
      <Form.Item label="名称" name="name" rules={[{ required: true }]}>
        <Input />
      </Form.Item>
      <Button type="primary">提交</Button>
    </Form>
  );
}
```

## 自定义样式范围

如需定制，建议使用组件的 className 和 style 属性，范围限定到局部区域，不覆盖全局样式。
