# 后端功能清单

本文档列出 openDeepWiki 项目的所有后端功能模块，用于测试用例编制。

## 1. 仓库管理 (Repository Management)

### 1.1 仓库创建
- **端点**: `POST /api/repositories`
- **功能**: 创建新的代码仓库
- **输入参数**:
  - `url`: 仓库地址
  - `name`: 仓库名称
- **测试要点**:
  - 成功创建仓库
  - 重复URL返回409错误
  - 无效URL返回400错误

### 1.2 仓库列表
- **端点**: `GET /api/repositories`
- **功能**: 获取所有仓库列表
- **测试要点**:
  - 返回完整列表
  - 空列表处理

### 1.3 仓库详情
- **端点**: `GET /api/repositories/:id`
- **功能**: 获取指定仓库详情
- **测试要点**:
  - 存在的仓库返回详情
  - 不存在的仓库返回404
  - 无效ID格式返回400

### 1.4 仓库删除
- **端点**: `DELETE /api/repositories/:id`
- **功能**: 删除指定仓库
- **测试要点**:
  - 成功删除仓库
  - 无效状态返回400错误
  - 不存在的仓库返回404

### 1.5 运行所有任务
- **端点**: `POST /api/repositories/:id/run-all`
- **功能**: 启动仓库下所有任务
- **测试要点**:
  - 成功启动任务
  - 无效仓库ID返回400

### 1.6 重新克隆仓库
- **端点**: `POST /api/repositories/:id/clone`
- **功能**: 重新下载仓库
- **测试要点**:
  - 成功启动克隆
  - 状态校验

### 1.7 清空本地目录
- **端点**: `POST /api/repositories/:id/purge-local`
- **功能**: 删除仓库本地目录
- **测试要点**:
  - 成功清空本地目录

### 1.8 目录分析
- **端点**: `POST /api/repositories/:id/directory-analyze`
- **功能**: 触发目录结构分析
- **测试要点**:
  - 成功发布分析事件

### 1.9 数据库模型分析
- **端点**: `POST /api/repositories/:id/db-model-analyze`
- **功能**: 触发数据库模型分析
- **测试要点**:
  - 成功发布分析事件

### 1.10 API接口分析
- **端点**: `POST /api/repositories/:id/api-analyze`
- **功能**: 触发API接口分析
- **测试要点**:
  - 成功发布分析事件

### 1.11 增量分析
- **端点**: `POST /api/repositories/:id/incremental-analysis`
- **功能**: 触发增量分析
- **测试要点**:
  - 成功发布分析事件

### 1.12 设置就绪状态
- **端点**: `POST /api/repositories/:id/set-ready`
- **功能**: 设置仓库为就绪状态
- **测试要点**:
  - 成功设置状态

### 1.13 获取增量历史
- **端点**: `GET /api/repositories/:id/incremental-history`
