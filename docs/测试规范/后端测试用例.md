### 1.1 仓库创建 (POST /api/repositories)

#### 测试用例 1.1.1: 成功创建仓库
```go
func TestRepositoryHandler_Create_Success(t *testing.T) {
    // Arrange
    reqBody := map[string]string{"url": "https://github.com/test/repo", "name": "test-repo"}
    req := httptest.NewRequest("POST", "/api/repositories", jsonReader(reqBody))
    req.Header.Set("Content-Type", "application/json")
    w := httptest.NewRecorder()

    // Act
    handler.Create(w, req)

    // Assert
    assert.Equal(t, http.StatusCreated, w.Code)
    var repo Repository
    json.Unmarshal(w.Body.Bytes(), &repo)
    assert.Equal(t, "https://github.com/test/repo", repo.URL)
    assert.Equal(t, "test-repo", repo.Name)
    assert.NotZero(t, repo.ID)
}
```

#### 测试用例 1.1.2: 重复URL返回409错误
