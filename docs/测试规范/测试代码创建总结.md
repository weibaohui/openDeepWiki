## 四、测试用例分布

### 4.1 后端测试用例分类

#### Handler层测试 (Handler Layer)
- 仓库管理：创建、列表、详情、删除、克隆、清空、运行、分析等
- 任务管理：获取、排队、运行、重试、取消、删除等
- 文档管理：获取、更新、评分、导出、版本等
- API Key管理：创建、列表、详情、更新、删除、状态更新、统计
- 同步管理：启动、拉取、状态、目标管理等
- 用户需求：创建、列表、详情、删除、状态更新
- 活跃度配置：获取、更新

#### 测试类型
- **成功场景测试**: 验证正常业务流程
- **参数验证测试**: 验证必填字段、格式校验
- **错误处理测试**: 验证404、409、400等错误码
- **边界条件测试**: 验证参数边界值

### 4.2 前端测试用例分类

#### 组件测试 (Component Tests)
- **渲染测试**: 验证组件正确渲染
- **交互测试**: 验证用户交互行为
- **状态测试**: 验证组件状态变化
- **响应式测试**: 验证不同屏幕尺寸适配

#### 服务层测试 (Service Tests)
- **API调用测试**: 验证API请求/响应
- **错误处理测试**: 验证网络错误处理
- **数据转换测试**: 验证响应数据转换
